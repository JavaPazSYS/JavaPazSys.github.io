Experiencia & Enfoque

Experiencia

Inicié mi carrera desarrollando automatizaciones con UiPath mientras finalizaba la universidad, trabajando con lógica de negocio, integración de sistemas y colaboración con equipos técnicos.

Con el tiempo amplié mi enfoque hacia el desarrollo de aplicaciones web, trabajando tanto en la lógica del sistema como en la gestión de datos con Java, Python y SQL.

Dentro de ese camino descubrí un especial interés por el frontend: el punto donde la lógica se encuentra con la experiencia del usuario. Disfruto diseñar y construir interfaces claras, coherentes y bien estructuradas.

Busco desarrollar soluciones completas donde arquitectura, lógica y experiencia visual funcionen como un sistema integrado.

Enfoque profesional

Mi forma de trabajo parte desde la ingeniería, no desde el código.

Primero analizo el problema, investigo el contexto y planteo una hipótesis de solución. A partir de ahí defino la arquitectura del sistema: estructura de datos, interacción entre componentes, flujo de pantallas e identidad visual.

Cuando la estructura está clara, comienza el desarrollo. De esta forma el código se convierte en la implementación de una solución previamente pensada, permitiendo construir sistemas claros, mantenibles y escalables.

Porque antes que informática, está la ingeniería.
